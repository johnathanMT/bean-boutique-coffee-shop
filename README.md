<h2 id="english">🇬🇧 English</h2>

* **Interactive Navigation:** Features a dynamic and responsive navbar.

### ✨ 主な機能 (Key Features)
* **完全なレスポンシブ対応 (Fully Responsive):** モバイル、タブレット、デスクトップなど、あらゆるデバイスの画面サイズに最適化され、シームレスなUIを提供します。
* **インタラクティブなナビゲーション (Interactive Navigation):** モバイルでのトグル機能やデスクトップでの表示切替に対応した、動的でレスポンシブなナビゲーションバーを実装しています。
* **モダンなUI/UX (Modern UI/UX):** 高級コーヒーショップのブランドイメージに合わせた、清潔感のある美しさと読みやすいタイポグラフィを採用しています。
* **パフォーマンスの最適化 (Optimized Performance):** 重いフレームワークに依存せず、HTML, CSS, JavaScriptのみを用いた軽量な構造で構築されています。

### 🛠️ 使用技術 (Technologies Used)
* **HTML5:** アクセシビリティとSEOを意識したセマンティックなマークアップ。
* **CSS3:** カスタムスタイリング、Flexbox/Gridを用いたレイアウト、およびメディアクエリによるレスポンシブ対応。
* **JavaScript (Vanilla):** インタラクティブな要素（モバイルメニューなど）のためのDOM操作。
