# ☕ Bean Boutique Coffee Shop

A modern, responsive, and visually appealing front-end web application for a boutique coffee shop. This project serves as a demonstration of clean UI/UX design principles and responsive web development using core web technologies.

## ✨ Key Features

* **Fully Responsive Design:** Seamlessly adapts to all device sizes (Mobile, Tablet, Desktop) ensuring a perfect viewing experience.
* **Interactive Navigation:** Features a dynamic and responsive navbar that handles desktop visibility and mobile toggling effectively.
* **Modern UI/UX:** Clean aesthetics, readable typography, and inviting color schemes tailored for a premium coffee shop brand.
* **Optimized Performance:** Lightweight structure built purely with HTML, CSS, and JS without heavy frameworks.

## 🛠️ Technologies Used

* **HTML5:** Semantic markup for better accessibility and SEO.
* **CSS3:** Custom styling, Flexbox/Grid for layout, and media queries for responsiveness.
* **JavaScript (Vanilla):** DOM manipulation for interactive elements (like the mobile menu).
